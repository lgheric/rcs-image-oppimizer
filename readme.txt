== Installation ==

Installation from within WordPress
Visit Plugins > Add New.
Search for "RCS Image Optimizer".
Install and activate the "RCS Image Optimizer" plugin.

Manual installation
Upload the entire rcs-image-optimizer folder to the /wp-content/plugins/ directory.
Visit Plugins.
Activate the "RCS Image Optimizer" plugin.
